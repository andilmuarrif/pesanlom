Bantuan

Halaman ini akan menuntun anda untuk mempermudah dalam menggunakan aplikasi ini. Berikut tata cara untuk penggunaan aplikasi Pesanlom :

1. Klik button Login, maka anda akan dialihkan ke halaman Login user dimana kita diberikan 2 pilihan yaitu sebagai user atau manager, pilih salah satunya.

2. Setelah itu, kita akan dialihkan ke halaman Login berdasarkan user pilihan kita

3. Masukkan username dan password dengan benar.

4. Klik OK. Maka anda akan login ke aplikasi Pesanlom.
